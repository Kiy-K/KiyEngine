Features

What makes KiyEngine different.

‚ö° BitNet Inference

Ternary {-1, 0, +1} weights eliminate floating-point multiply. Just additions and subtractions at the core.

üîç Alpha-Beta Search

Full iterative deepening with aspiration windows, null move pruning, LMR, SEE pruning, and PV tracking.

üßµ Lazy SMP

Multi-threaded parallel search with shared transposition table. Helper threads use varied depth offsets.

üìñ Opening Book

Built-in binary opening book with position hashing for fast lookup. Plays known theory before switching to search.

‚è± Smart Time Management

Stockfish-inspired soft/hard bounds with dynamic adjustment for falling eval and best-move instability.

üîå Full UCI Protocol

Compatible with Arena, Cutechess, lichess-bot, and any UCI chess GUI. Hash, Threads, MultiPV, and more.

Quick Start

Get KiyEngine running in under a minute.

1

Download

Grab the binary for your platform from the downloads section above.

2

Extract (Linux)

Run tar -xzf kiyengine-linux-x86_64.tar.gz and chmod +x kiyengine

3

Run

Launch ./kiyengine (Linux) or kiyengine.exe (Windows). It speaks UCI ‚Äî type uci to verify.

4

Use with a GUI

Load as a UCI engine in Cutechess, Arena, or lichess-bot.

Build from Source

Requires Rust 1.75+ and a C compiler for candle/libtorch.

$

git clone https://github.com/Kiy-K/KiyEngine.git

cd KiyEngine && cargo build --release

./target/release/kiy_engine_v5_alpha
